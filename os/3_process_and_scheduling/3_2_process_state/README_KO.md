# 🔑 process state

<br>

## 📌 멀티 프로그밍과 Wait

● 멀티 프로그밍: CPU 활용도를 극대화 하는 스케쥴링 알고리즘<br>
● Wait: 간단히 저장매체로부터 파일 읽기를 기다리는 시간으로 지정<br>

<br>

## 📌 프로세스 상태

● running state: 현재 CPU에서 실행 상태<br>
● ready state: CPU에서 실행 가능 상태(실행 대기 상태)<br>
● block state: 특정 이벤트 발생 대기 상태(예: 프린팅이 다되었다!)<br>
